# BoxHero Desktop App

[박스히어로](https://www.boxhero-app.com) 데스크톱 앱

![제품목록](screenshots/item_list.png)

## NPM Scripts

### 설치

```sh
yarn install
```

### 개발

```sh
yarn start
```

### 패키징 & 배포

```sh
yarn package
yarn make
yarn publish
```
